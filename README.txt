QR Code Reader - Packaged Web App

Files included:
- index.html: Main QR Code Reader app
- manifest.json: Installable PWA manifest
- service-worker.js: Basic cache/service worker file
- icons/: App icons

How to use:
1. Upload all files to GitHub Pages, Netlify, Vercel, or any HTTPS hosting.
2. Open index.html in the browser.
3. Allow camera permission.
4. Use the Install App button or browser install option where supported.

Important:
Camera access usually requires HTTPS, except on localhost. If opened directly from file manager, the camera may not work.
