QR Code Reader - Packaged PWA

This package includes an improved SGV QR-style logo, app icons, manifest.json, and service-worker.js.

How to use:
1. Upload the full folder contents to GitHub Pages, Netlify, Vercel, or any HTTPS hosting.
2. Open index.html from the hosted link.
3. Allow camera permission.
4. Use the Install App button when your browser shows it.

Important:
Camera scanning usually requires HTTPS. Opening the file directly from your phone storage may not allow camera access.

Updated logo: The package uses the uploaded QR scanner image as logo.png and app icons.
